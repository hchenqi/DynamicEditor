## Build

This project can be built with CMake on Windows. It depends on two other major projects of mine, *BlockStore* and *ViewDesign*.

## Implementation

An initial research about dynamic typing is carried out and documented in [Preparation](docs/preparation.md), which includes an implementation prototype focusing on the data layer.

A more complete design and implementation plan based on and simplified from the prototype can be found in [Design](docs/design.md), which includes the type definitions for both the data and the view layer.

The final draft of the implementation can be found in [Implementation](docs/implementation.md), which outlines the source code structure.
